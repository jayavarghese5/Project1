# Project Product_data 
This project takes a csv file with 5 columns and adds the contents in the db.

Prerequisites
java 1.8
mysql

To Run this project 

1)open the cmd prompt and run the command java -jar upload-csv-mysql.jar
2) open the browser and go to http://localhost:8080/
3) upload the csv file (sample csv file is present in the uploads folder of the project)
